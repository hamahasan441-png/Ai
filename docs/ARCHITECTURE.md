## Key Design Decisions

1. **Zero external dependencies for infrastructure** вҖ” Auth, API, database, config all use Node.js built-ins
2. **In-memory-first** вҖ” All stores (database, cache, checkpoints) work in-memory with optional persistence
3. **Plugin-based extensibility** вҖ” 11 permission types, lifecycle hooks, per-plugin storage
4. **Resilience patterns** вҖ” Retry with exponential backoff, circuit breaker, dead letter queue
5. **Multi-model support** вҖ” Unified LLM interface supporting Claude, OpenAI, and Ollama

## Technology Stack

- **Runtime**: Node.js 18/20/22
- **Language**: TypeScript (strict mode)
- **Testing**: Vitest with v8 coverage
- **Linting**: ESLint + Prettier
- **CI/CD**: GitHub Actions (lint, format, typecheck, test, coverage, Docker, security audit)
